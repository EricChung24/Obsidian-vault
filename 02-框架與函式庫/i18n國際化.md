# i18n 國際化

> 讓你的應用支援多語言，觸及更廣的用戶群。

← [[前端工程師]] | 相關：[[React]] | [[Vue]] | [[Next.js]]

---

## 核心概念

```
i18n = internationalization（18 是中間省略的字母數）
l10n = localization（在地化）

翻譯鍵（Translation Key）
   ↓
翻譯檔（Locale File）
   ↓
渲染時依當前語系替換文字
```

### 初始化設定

```ts
// src/i18n.ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import LanguageDetector from 'i18next-browser-languagedetector';

import zhTW from './locales/zh-TW/translation.json';
import en   from './locales/en/translation.json';

i18n
  .use(LanguageDetector)    // 自動偵測瀏覽器語系
  .use(initReactI18next)
  .init({
    resources: {
      'zh-TW': { translation: zhTW },
      'en':    { translation: en },
    },
    fallbackLng: 'zh-TW',   // 找不到翻譯時的備援語系
    interpolation: {
      escapeValue: false,   // React 已處理 XSS，不需要跳脫
    },
    detection: {
      order: ['localStorage', 'navigator'],  // 偵測順序
      caches: ['localStorage'],
    },
  });

export default i18n;
```

```tsx
// src/main.tsx
import './i18n';  // 在 App 之前引入
import App from './App';
```

### 在元件中使用

```tsx
import { useTranslation } from 'react-i18next';

function Header() {
  const { t, i18n } = useTranslation();

  const switchLang = (lang: string) => {
    i18n.changeLanguage(lang);
  };

  return (
    <header>
      <nav>
        <a href="/">{t('nav.home')}</a>
        <a href="/about">{t('nav.about')}</a>
      </nav>

      {/* 語言切換按鈕 */}
      <div>
        <button
          onClick={() => switchLang('zh-TW')}
          className={i18n.language === 'zh-TW' ? 'active' : ''}
        >
          繁中
        </button>
        <button
          onClick={() => switchLang('en')}
          className={i18n.language === 'en' ? 'active' : ''}
        >
          EN
        </button>
      </div>
    </header>
  );
}

function WelcomePage({ user }) {
  const { t } = useTranslation();

  return (
    <div>
      {/* 插值 */}
      <h1>{t('auth.welcome', { name: user.name })}</h1>

      {/* 複數形式 */}
      <p>{t('product.count', { count: 42 })}</p>
      <p>{t('product.count', { count: 1 })}</p>

      {/* 命名空間（多個翻譯檔）*/}
      {/* const { t } = useTranslation('profile'); */}
    </div>
  );
}
```

## Vue：vue-i18n

```bash
pnpm add vue-i18n@9
```

```ts
// src/i18n.ts
import { createI18n } from 'vue-i18n';

const messages = {
  'zh-TW': {
    nav: { home: '首頁', about: '關於' },
    greeting: '哈囉，{name}！',
    items: '沒有項目 | 一個項目 | {count} 個項目',
  },
  en: {
    nav: { home: 'Home', about: 'About' },
    greeting: 'Hello, {name}!',
    items: 'no items | one item | {count} items',
  },
};

export const i18n = createI18n({
  legacy: false,          // 使用 Composition API 模式
  locale: 'zh-TW',
  fallbackLocale: 'en',
  messages,
});
```

```ts
// main.ts
import { createApp } from 'vue';
import { i18n } from './i18n';
import App from './App.vue';

createApp(App).use(i18n).mount('#app');
```

```vue
<!-- 在 Vue 元件中使用 -->
<template>
  <div>
    <!-- 模板語法 -->
    <h1>{{ $t('nav.home') }}</h1>
    <p>{{ $t('greeting', { name: 'Alice' }) }}</p>

    <!-- 複數 -->
    <p>{{ $tc('items', 0) }}</p>   <!-- 沒有項目 -->
    <p>{{ $tc('items', 1) }}</p>   <!-- 一個項目 -->
    <p>{{ $tc('items', 5) }}</p>   <!-- 5 個項目 -->

    <!-- 語言切換 -->
    <button @click="$i18n.locale = 'zh-TW'">繁中</button>
    <button @click="$i18n.locale = 'en'">EN</button>
  </div>
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n';

const { t, locale } = useI18n();

// Composition API 用法
console.log(t('nav.home'));
locale.value = 'en';  // 切換語系
</script>
```

## Next.js 國際化（App Router）

```js
// next.config.js
module.exports = {
  i18n: {
    locales: ['zh-TW', 'en', 'ja'],
    defaultLocale: 'zh-TW',
  },
};
```

```
網址結構：
/           → zh-TW（預設語系不加前綴）
/en         → English
/ja         → 日本語
/en/about   → About（英文版）
```

```tsx
// app/[lang]/layout.tsx
export async function generateStaticParams() {
  return [{ lang: 'zh-TW' }, { lang: 'en' }, { lang: 'ja' }];
}

// 載入翻譯（搭配 server-i18n 或直接 import JSON）
async function getDictionary(lang: string) {
  return (await import(`./dictionaries/${lang}.json`)).default;
}

export default async function Layout({
  children,
  params: { lang },
}: {
  children: React.ReactNode;
  params: { lang: string };
}) {
  const dict = await getDictionary(lang);
  return (
    <html lang={lang}>
      <body>{children}</body>
    </html>
  );
}
```

## 日期/數字在地化

```js
// Intl API（原生，不需額外函式庫）

// 日期格式化
const date = new Date('2026-04-16');

new Intl.DateTimeFormat('zh-TW', {
  year: 'numeric', month: 'long', day: 'numeric'
}).format(date);
// '2026年4月16日'

new Intl.DateTimeFormat('en-US', {
  year: 'numeric', month: 'short', day: 'numeric'
}).format(date);
// 'Apr 16, 2026'

// 數字格式化
new Intl.NumberFormat('zh-TW', {
  style: 'currency', currency: 'TWD', minimumFractionDigits: 0,
}).format(12345);
// 'NT$12,345'

new Intl.NumberFormat('en-US', {
  style: 'currency', currency: 'USD',
}).format(12345.678);
// '$12,345.68'

// 相對時間
const rtf = new Intl.RelativeTimeFormat('zh-TW', { numeric: 'auto' });
rtf.format(-1, 'day');   // '昨天'
rtf.format(-7, 'day');   // '7 天前'
rtf.format(1, 'week');   // '下週'
```

## 最佳實踐

```
✅ 翻譯鍵用語意命名：auth.loginButton，不用 btn1
✅ 建立翻譯鍵的型別（TypeScript）避免拼錯
✅ 預設語系設定 fallbackLng，避免白畫面
✅ 數字/日期用 Intl API，不要寫死格式
✅ 考慮 RTL（阿拉伯/希伯來語）：dir="rtl"
✅ 翻譯字串預留長度空間（英文比中文長 1.5~2 倍）
```
