## 盒模型（Box Model）

每個 HTML 元素都是一個盒子，由四個層次組成。

```
┌──────────────────────────────┐
│          margin（外距）        │
│  ┌────────────────────────┐  │
│  │      border（邊框）      │  │
│  │  ┌──────────────────┐  │  │
│  │  │  padding（內距）   │  │  │
│  │  │  ┌────────────┐  │  │  │
│  │  │  │  content   │  │  │  │
│  │  │  └────────────┘  │  │  │
│  │  └──────────────────┘  │  │
│  └────────────────────────┘  │
└──────────────────────────────┘
```

```css
/* box-sizing 的重要性 */

/* 預設 content-box：width = 內容寬度，不含 padding/border */
.default {
  box-sizing: content-box;
  width: 200px;
  padding: 20px;
  border: 2px solid;
  /* 實際佔寬 = 200 + 40 + 4 = 244px */
}

/* border-box（推薦）：width = 總寬度，含 padding/border */
.better {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 2px solid;
  /* 實際佔寬 = 200px（直覺！）*/
}

/* 全域套用（現代最佳實踐）*/
*, *::before, *::after {
  box-sizing: border-box;
}
```

## 定位（Position）

```css
/* static（預設）— 正常文件流 */
.normal { position: static; }

/* relative — 相對自身原位移動，仍佔據原空間 */
.badge {
  position: relative;
  top: -4px;     /* 往上移 4px */
  left: 8px;
}

/* absolute — 脫離文件流，相對最近的 positioned 祖先定位 */
/* 常見用法：absolute 放在 relative 父容器內 */
.card {
  position: relative;  /* 建立定位基準 */
}
.card__badge {
  position: absolute;
  top: 8px;
  right: 8px;
}

/* fixed — 相對瀏覽器視窗定位，捲動不移動 */
.navbar {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  z-index: 100;
}

/* sticky — 捲動到特定位置時「黏住」 */
.section-title {
  position: sticky;
  top: 64px;       /* 距頂部 64px 時開始黏住 */
  background: white;
  z-index: 10;
}
```

### 常見：absolute 置中技巧

```css
/* 方法一：transform（推薦，不需知道寬高）*/
.overlay {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 方法二：inset + margin auto（現代）*/
.overlay {
  position: absolute;
  inset: 0;          /* top/right/bottom/left 全為 0 */
  margin: auto;
  width: fit-content;
  height: fit-content;
}
```

## Flexbox 切版

### 核心概念

```css
.container {
  display: flex;

  /* 主軸方向 */
  flex-direction: row;            /* → 橫排（預設）*/
  flex-direction: column;         /* ↓ 直排 */
  flex-direction: row-reverse;    /* ← 反向橫排 */

  /* 主軸對齊 */
  justify-content: flex-start;    /* 靠左 */
  justify-content: center;        /* 置中 */
  justify-content: flex-end;      /* 靠右 */
  justify-content: space-between; /* 均分，兩端無留白 */
  justify-content: space-around;  /* 均分，兩端有半留白 */
  justify-content: space-evenly;  /* 完全均分 */

  /* 交叉軸對齊 */
  align-items: stretch;    /* 延伸填滿（預設）*/
  align-items: center;     /* 垂直置中 */
  align-items: flex-start; /* 靠上 */
  align-items: flex-end;   /* 靠下 */
  align-items: baseline;   /* 文字基線對齊 */

  /* 換行 */
  flex-wrap: nowrap;   /* 不換行（預設）*/
  flex-wrap: wrap;     /* 超出時換行 */

  gap: 1rem;           /* 子項間距（row-gap + column-gap）*/
}
```

### 常用版型

```css
/* ── 1. 導覽列（Logo 左 / 選單右）── */
.navbar {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 0 1.5rem;
  height: 64px;
}

/* ── 2. 垂直水平置中 ── */
.hero {
  display: flex;
  align-items: center;
  justify-content: center;
  min-height: 100vh;
}

/* 或更簡潔 */
.hero {
  display: grid;
  place-items: center;   /* align + justify 一次搞定 */
  min-height: 100vh;
}

/* ── 3. 卡片等高 ── */
.card-list {
  display: flex;
  flex-wrap: wrap;
  gap: 1.5rem;
}
.card {
  flex: 1 1 280px;   /* 最小 280px，能伸能縮 */
  display: flex;
  flex-direction: column;
}
.card__body {
  flex: 1;           /* 撐滿剩餘空間，讓按鈕永遠在底部 */
}

/* ── 4. Sidebar + Main（舊方案）── */
.layout {
  display: flex;
  min-height: 100vh;
}
.sidebar {
  flex: 0 0 240px;   /* 固定 240px，不縮不長 */
}
.main {
  flex: 1;           /* 佔用所有剩餘空間 */
}
```

## CSS Grid 切版

### 核心概念

```css
.grid {
  display: grid;

  /* 定義欄數 */
  grid-template-columns: 240px 1fr 1fr;           /* 固定 + 彈性 */
  grid-template-columns: repeat(3, 1fr);           /* 3 等分 */
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); /* 自動換行 */

  /* 定義列高 */
  grid-template-rows: auto;
  grid-template-rows: 64px 1fr 80px;              /* Header + Main + Footer */

  gap: 1rem;
  gap: 1rem 2rem;    /* row-gap column-gap */
}
```

### 命名區域（直覺排版）

```css
/* ── 頁面整體版型 ── */
.page {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  grid-template-columns: 240px 1fr;
  grid-template-rows: 64px 1fr 80px;
  min-height: 100vh;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }
```

```html
<div class="page">
  <header class="header">Logo + Nav</header>
  <aside class="sidebar">側欄選單</aside>
  <main class="main">主要內容</main>
  <footer class="footer">版權資訊</footer>
</div>
```

### 自動響應式卡片格

```css
/* 不需要 media query！*/
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 1.5rem;
}
```

```
320px 寬：  [  card  ]
640px 寬：  [card][card]
960px 寬：  [card][card][card]
1280px 寬：[card][card][card][card]
```

### 跨欄跨列

```css
.featured {
  grid-column: 1 / 3;    /* 從第 1 條線到第 3 條線（佔 2 欄）*/
  grid-row: 1 / 3;       /* 佔 2 列 */
}

/* 簡寫 */
.featured {
  grid-column: span 2;   /* 佔 2 欄（不指定起始位置）*/
  grid-row: span 2;
}
```

## 響應式設計（RWD）

### Mobile First 思維

```css
/* 預設寫手機版，用 min-width 逐步升級 */

.container {
  padding: 1rem;
}

/* 平板（768px+）*/
@media (min-width: 768px) {
  .container {
    padding: 2rem;
  }
}

/* 桌機（1024px+）*/
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 3rem 2rem;
  }
}
```

### 響應式導覽列

```css
.nav {
  display: flex;
  align-items: center;
  justify-content: space-between;
}

/* 手機：隱藏選單，顯示漢堡按鈕 */
.nav__links {
  display: none;
}
.nav__burger {
  display: block;
}

/* 桌機：顯示選單，隱藏漢堡 */
@media (min-width: 768px) {
  .nav__links {
    display: flex;
    gap: 2rem;
  }
  .nav__burger {
    display: none;
  }
}
```

### 響應式文字

```css
/* clamp(最小值, 理想值, 最大值) — 不需要 media query */
h1 {
  font-size: clamp(1.5rem, 4vw, 3rem);
}
p {
  font-size: clamp(0.875rem, 2vw, 1.125rem);
}

/* 響應式間距 */
.section {
  padding: clamp(2rem, 8vw, 6rem) clamp(1rem, 5vw, 3rem);
}
```

## 常見切版問題 & 解法

### 問題 1：子元素超出父容器

```css
/* 問題：長文字或圖片撐破容器 */
img { max-width: 100%; }          /* 圖片不超出 */
p { overflow-wrap: break-word; }   /* 長字串自動換行 */
```

### 問題 2：Flexbox 子項目不等高

```css
/* 問題：卡片高度不一致 */
/* 解法：用 flex column，讓 body 撐滿 */
.card {
  display: flex;
  flex-direction: column;
}
.card__content {
  flex: 1;  /* 撐滿，讓 footer 貼底 */
}
```

### 問題 3：Margin Collapse（垂直 margin 合併）

```css
/* 問題：上下相鄰元素的 margin 會合併取大值 */
.a { margin-bottom: 20px; }
.b { margin-top: 30px; }
/* 實際間距 = 30px（不是 50px！）*/

/* 解法一：只設單方向 margin */
* + * { margin-top: 1rem; }   /* Lobotomized Owl */

/* 解法二：父元素建立 BFC */
.parent {
  overflow: hidden;   /* 觸發 BFC */
  /* 或 display: flex/grid */
}

/* 解法三：改用 gap（Flex/Grid）*/
.list {
  display: flex;
  flex-direction: column;
  gap: 1rem;            /* 不會有 collapse 問題 */
}
```

### 問題 4：Fixed 定位的遮擋問題

```css
/* 固定 navbar 會遮住內容頂部 */
body {
  padding-top: 64px;   /* 等於 navbar 高度 */
}

/* 或用 scroll-margin-top（錨點跳轉時）*/
section {
  scroll-margin-top: 72px;
}
```

### 問題 5：Sticky 不生效

```css
/* sticky 需要父容器有高度且 overflow 不是 hidden/auto */
.sticky-header {
  position: sticky;
  top: 0;
}

/* 常見原因：
   1. 父元素 overflow: hidden → 改用 clip
   2. 父元素沒有足夠高度
   3. 沒有設 top/left/right/bottom 其中一個 */
```

## 實用 CSS Reset

```css
/* 現代 CSS Reset */
*, *::before, *::after {
  box-sizing: border-box;
}

* {
  margin: 0;
  padding: 0;
}

html {
  font-size: 16px;
  scroll-behavior: smooth;
}

body {
  line-height: 1.5;
  -webkit-font-smoothing: antialiased;
}

img, picture, video, canvas, svg {
  display: block;
  max-width: 100%;
}

input, button, textarea, select {
  font: inherit;    /* 繼承 body 字型 */
}

p, h1, h2, h3, h4, h5, h6 {
  overflow-wrap: break-word;
}

/* 無障礙：讓焦點樣式清晰 */
:focus-visible {
  outline: 2px solid #3b82f6;
  outline-offset: 2px;
}
```
