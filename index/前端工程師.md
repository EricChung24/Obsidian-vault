## 建議學習路線

1. 先打穩：`HTML`、`CSS`、`JavaScript`、`Git`
2. 再進框架：`TypeScript`、`React`、`React Router`、`表單處理`
3. 補工程能力：`React Query`、`測試策略`、`Vite(主流)`、`前端快取策略`
4. 補底層：`Event Loop`、`HTTP協議`、`CORS`、`Cookie、Session、JWT`
5. 補進階：`React效能優化`、`渲染優化`、`CI-CD`、`前端監控`
6. 補現代技術：`React Server Components`、`GraphQL`、`WebSocket與即時通訊`、`現代CSS新特性`、`Monorepo工具`、`Framer Motion`
