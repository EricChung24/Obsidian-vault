# a11y 無障礙設計

> 讓所有人（含視障、肢障、認知障礙者）都能使用你的網站。

← [[前端工程師]]

---

## WCAG 四大原則

| 原則 | 說明 |
|------|------|
| **P** erceivable（可感知）| 內容要能被感知，如 alt 文字、字幕 |
| **O** perable（可操作）| 所有功能可用鍵盤操作 |
| **U** nderstandable（可理解）| 語言清晰、行為可預期 |
| **R** obust（穩健）| 與輔助技術相容 |

## 語意化 HTML（最重要）

```html
<!-- 不要這樣 -->
<div class="btn" onclick="submit()">送出</div>

<!-- 要這樣 -->
<button type="submit">送出</button>

<!-- 正確使用標題層級 -->
<h1>網站標題</h1>        <!-- 每頁只有一個 -->
  <h2>主要區塊</h2>
    <h3>子區塊</h3>
    <h3>子區塊</h3>
  <h2>另一個主要區塊</h2>

<!-- 表單標籤 -->
<label for="name">姓名</label>
<input id="name" type="text" autocomplete="name">
```

## ARIA 屬性

```html
<!-- 角色 -->
<div role="dialog" aria-modal="true" aria-labelledby="dialog-title">
  <h2 id="dialog-title">確認刪除</h2>
  <button aria-label="關閉對話框">✕</button>
</div>

<!-- 狀態 -->
<button aria-expanded="false" aria-controls="menu">選單</button>
<nav id="menu" aria-hidden="true">...</nav>

<!-- 即時區域（動態更新通知） -->
<div role="status" aria-live="polite">已儲存</div>
<div role="alert" aria-live="assertive">錯誤！請重試</div>

<!-- 進度條 -->
<div
  role="progressbar"
  aria-valuenow="65"
  aria-valuemin="0"
  aria-valuemax="100"
  aria-label="上傳進度"
>
  65%
</div>
```

## 鍵盤導航

```tsx
// 焦點管理（Modal 開啟時）
function Modal({ isOpen, onClose }) {
  const modalRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (isOpen) {
      // 開啟時聚焦
      modalRef.current?.focus();
    }
  }, [isOpen]);

  // 按 Esc 關閉
  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'Escape') onClose();

    // 焦點陷阱 (Focus Trap)
    if (e.key === 'Tab') {
      const focusableElements = modalRef.current?.querySelectorAll(
        'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
      );
      // 處理首尾循環...
    }
  };

  return isOpen ? (
    <div
      ref={modalRef}
      role="dialog"
      aria-modal="true"
      tabIndex={-1}
      onKeyDown={handleKeyDown}
    >
      {/* 內容 */}
    </div>
  ) : null;
}
```

## 色彩對比

| 等級 | 一般文字 | 大型文字（18px+）|
|------|---------|----------------|
| AA（最低標準）| 4.5:1 | 3:1 |
| AAA（最高標準）| 7:1 | 4.5:1 |

```css
/* 不要只用顏色傳遞資訊 */
/* 錯誤 */
.error { color: red; }

/* 正確：加上圖示或底線 */
.error {
  color: #dc2626;
  border-left: 4px solid currentColor;
}
.error::before {
  content: '⚠ ';
}
```

## Skip Link（跳過導覽）

```html
<a href="#main-content" class="skip-link">跳到主內容</a>
<nav>...</nav>
<main id="main-content">...</main>
```

```css
.skip-link {
  position: absolute;
  left: -9999px;
}
.skip-link:focus {
  left: 0;
  top: 0;
  z-index: 9999;
  padding: 0.5rem;
  background: #000;
  color: #fff;
}
```
