# 正則表達式（Regex）

> 正則表達式是處理字串的「超級武器」，掌握它讓你寫出簡潔高效的驗證與解析邏輯。

← [[前端工程師]] | 相關：[[JavaScript]] | [[TypeScript]]

---

## 基本語法

```js
// 兩種建立方式
const re1 = /pattern/flags;          // 字面量（推薦，效能好）
const re2 = new RegExp('pattern', 'flags');  // 動態建立（pattern 可為變數）
```

### 常用 Flags

| Flag | 說明 |
|------|------|
| `g`  | Global — 找出所有符合，不只第一個 |
| `i`  | Ignore case — 忽略大小寫 |
| `m`  | Multiline — `^` `$` 匹配每行頭尾 |
| `s`  | DotAll — `.` 也匹配換行符 |
| `u`  | Unicode — 完整 Unicode 支援（emoji 等）|

## 字元類別

```js
// 單一字元匹配
.        // 任意字元（除換行符）
\d       // 數字  [0-9]
\D       // 非數字
\w       // 字母/數字/底線  [a-zA-Z0-9_]
\W       // 非 \w
\s       // 空白（空格/tab/換行）
\S       // 非空白
\b       // 單字邊界
\B       // 非單字邊界

// 字元集合
[abc]    // a 或 b 或 c
[^abc]   // 不是 a、b、c
[a-z]    // a 到 z
[a-zA-Z0-9]  // 字母或數字
```

## 量詞（Quantifiers）

```js
*        // 0 次以上（貪婪）
+        // 1 次以上（貪婪）
?        // 0 或 1 次
{n}      // 恰好 n 次
{n,}     // 至少 n 次
{n,m}    // n 到 m 次

// 貪婪 vs 懶惰
/<.+>/   // 貪婪：匹配 <div>hello</div> 整個
/<.+?>/  // 懶惰：只匹配 <div>
```

## 群組與參考

```js
// 捕獲群組 (...)
const re = /(\d{4})-(\d{2})-(\d{2})/;
const match = '2026-04-16'.match(re);
// match[0] = '2026-04-16'（完整匹配）
// match[1] = '2026', match[2] = '04', match[3] = '16'

// 具名捕獲群組 (?<name>...)
const re2 = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const { year, month, day } = '2026-04-16'.match(re2).groups;
// year = '2026', month = '04', day = '16'

// 非捕獲群組 (?:...)
const re3 = /(?:https?|ftp):\/\//;  // 群組但不捕獲
```

## 常用方法

```js
const str = 'Hello World hello';
const re = /hello/gi;

// test() — 測試是否匹配，回傳 boolean
re.test('Hello World');  // true

// match() — 回傳匹配結果（無 g flag 時含群組）
str.match(/hello/i);   // ['Hello', index: 0, ...]
str.match(/hello/gi);  // ['Hello', 'hello']（g flag 時只回陣列）

// matchAll() — 取得所有匹配（含群組）
const iter = str.matchAll(/h(ello)/gi);
for (const m of iter) {
  console.log(m[0], m[1]);  // 'Hello' 'ello', 'hello' 'ello'
}

// replace() — 取代
str.replace(/hello/gi, 'Hi');  // 'Hi World Hi'
str.replace(/(\w+)\s(\w+)/, '$2 $1');  // 'World Hello hello'（交換前兩詞）

// replaceAll() — 全部取代（等同 g flag）
str.replaceAll('l', 'L');  // 'HeLLo WorLd heLLo'

// split() — 以正則分割
'a1b2c3'.split(/\d/);  // ['a', 'b', 'c', '']

// search() — 找到第一個匹配的 index（找不到為 -1）
str.search(/world/i);  // 6
```

## 前後斷言（Lookahead / Lookbehind）

```js
// 正向先行斷言 (?=...)：後面是...
/\d+(?= dollars)/.test('100 dollars');  // true，匹配 '100'
'100 dollars 200 euros'.match(/\d+(?= dollars)/g);  // ['100']

// 負向先行斷言 (?!...)：後面不是...
/\d+(?! dollars)/.exec('100 euros');  // 匹配 '100'

// 正向後行斷言 (?<=...)：前面是...（ES2018）
/(?<=\$)\d+/.exec('$500');  // 匹配 '500'

// 負向後行斷言 (?<!...)：前面不是...
/(?<!\$)\d+/.exec('€300');  // 匹配 '300'
```

## 實用範例集

```js
// ✅ Email 驗證
const emailRe = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
emailRe.test('user@example.com');  // true

// ✅ 台灣手機號碼（09 開頭，10 碼）
const phoneRe = /^09\d{8}$/;
phoneRe.test('0912345678');  // true

// ✅ 台灣身分證字號（英文字母+9位數字）
const idRe = /^[A-Z]\d{9}$/i;
idRe.test('A123456789');  // true

// ✅ 網址
const urlRe = /^https?:\/\/([\w-]+\.)+[\w-]+(\/[\w-./?%&=]*)?$/;
urlRe.test('https://www.google.com');  // true

// ✅ 密碼強度（8位以上，含大小寫+數字）
const pwRe = /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}$/;
pwRe.test('Password1');  // true

// ✅ 中文字元
const cnRe = /[\u4e00-\u9fa5]/;
cnRe.test('前端工程師');  // true

// ✅ 信用卡格式化（每4位加空格）
'4111111111111111'.replace(/(\d{4})(?=\d)/g, '$1 ');
// '4111 1111 1111 1111'

// ✅ 取出所有 HTML 標籤
'<div>Hello <span>World</span></div>'.match(/<[^>]+>/g);
// ['<div>', '<span>', '</span>', '</div>']

// ✅ 駝峰轉底線（camelCase → snake_case）
'camelCaseString'.replace(/([A-Z])/g, '_$1').toLowerCase();
// 'camel_case_string'

// ✅ 底線轉駝峰（snake_case → camelCase）
'snake_case_string'.replace(/_([a-z])/g, (_, c) => c.toUpperCase());
// 'snakeCaseString'

// ✅ 移除 HTML 標籤取純文字
'<p>Hello <strong>World</strong></p>'.replace(/<[^>]*>/g, '');
// 'Hello World'

// ✅ 金額格式化（加千分位）
'1234567'.replace(/\B(?=(\d{3})+(?!\d))/g, ',');
// '1,234,567'
```

## 動態正則（建構子用法）

```js
// 當 pattern 需要從變數產生時用 RegExp 建構子
function highlight(text, keyword) {
  // 注意：特殊字元需跳脫
  const escaped = keyword.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
  const re = new RegExp(`(${escaped})`, 'gi');
  return text.replace(re, '<mark>$1</mark>');
}

highlight('Hello World', 'world');
// 'Hello <mark>World</mark>'
```

## 效能注意事項

```js
// ❌ 在迴圈中重複建立正則（每次 new RegExp 都重新編譯）
for (const item of list) {
  if (new RegExp('^\\d+$').test(item)) { ... }  // 慢
}

// ✅ 提取到外部，只編譯一次
const numRe = /^\d+$/;
for (const item of list) {
  if (numRe.test(item)) { ... }  // 快
}

// ⚠️ 帶 g flag 的正則有狀態（lastIndex）
// 多次呼叫 test() 時要注意
const re = /\d+/g;
re.test('abc123');  // true（lastIndex 移到 6）
re.test('abc123');  // false（從 index 6 開始找，找不到）
// 解法：每次用新正則，或手動重設 re.lastIndex = 0;
```
