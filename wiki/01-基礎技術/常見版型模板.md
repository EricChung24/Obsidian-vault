## 版型 1：後台管理系統（Sidebar + Header + Main）

```html
<div class="admin-layout">
  <aside class="sidebar">
    <div class="sidebar__logo">LOGO</div>
    <nav class="sidebar__nav">
      <a href="#" class="nav-item nav-item--active">儀表板</a>
      <a href="#" class="nav-item">用戶管理</a>
      <a href="#" class="nav-item">設定</a>
    </nav>
  </aside>

  <div class="content-area">
    <header class="topbar">
      <h1 class="topbar__title">儀表板</h1>
      <div class="topbar__user">
        <img src="avatar.jpg" alt="用戶頭像" class="avatar">
        <span>Alice</span>
      </div>
    </header>

    <main class="main-content">
      <!-- 頁面內容 -->
    </main>
  </div>
</div>
```

```css
/* ── 後台管理版型 ── */
.admin-layout {
  display: flex;
  min-height: 100vh;
}

/* 側欄 */
.sidebar {
  width: 240px;
  flex-shrink: 0;          /* 不縮小 */
  background: #1e293b;
  color: white;
  display: flex;
  flex-direction: column;
  position: sticky;
  top: 0;
  height: 100vh;
  overflow-y: auto;
}

.sidebar__logo {
  padding: 1.5rem;
  font-size: 1.25rem;
  font-weight: 700;
  border-bottom: 1px solid rgba(255,255,255,0.1);
}

.sidebar__nav {
  display: flex;
  flex-direction: column;
  padding: 1rem 0;
}

.nav-item {
  padding: 0.75rem 1.5rem;
  color: #94a3b8;
  text-decoration: none;
  transition: all 0.2s;
}

.nav-item:hover,
.nav-item--active {
  background: rgba(255,255,255,0.1);
  color: white;
}

/* 右側內容區 */
.content-area {
  flex: 1;
  display: flex;
  flex-direction: column;
  min-width: 0;              /* 防止內容撐破 Flex */
}

/* 頂部 Header */
.topbar {
  height: 64px;
  padding: 0 2rem;
  background: white;
  border-bottom: 1px solid #e2e8f0;
  display: flex;
  align-items: center;
  justify-content: space-between;
  position: sticky;
  top: 0;
  z-index: 10;
}

.topbar__title {
  font-size: 1.25rem;
  font-weight: 600;
}

.topbar__user {
  display: flex;
  align-items: center;
  gap: 0.75rem;
}

.avatar {
  width: 36px;
  height: 36px;
  border-radius: 50%;
  object-fit: cover;
}

/* 主要內容 */
.main-content {
  flex: 1;
  padding: 2rem;
  background: #f8fafc;
}

/* 手機版：側欄折疊（需要 JS 控制 class）*/
@media (max-width: 768px) {
  .sidebar {
    position: fixed;
    left: -240px;
    z-index: 50;
    transition: left 0.3s ease;
  }
  .sidebar--open {
    left: 0;
  }
}
```

## 版型 3：登入表單（置中卡片）

```html
<div class="auth-page">
  <div class="auth-card">
    <div class="auth-card__header">
      <h1>歡迎回來</h1>
      <p>請輸入您的帳號密碼</p>
    </div>

    <form class="auth-form" novalidate>
      <div class="form-group">
        <label for="email" class="form-label">Email</label>
        <input
          type="email"
          id="email"
          class="form-input"
          placeholder="you@example.com"
          autocomplete="email"
          required
        >
        <span class="form-error" id="email-error" role="alert"></span>
      </div>

      <div class="form-group">
        <label for="password" class="form-label">
          密碼
          <a href="#" class="form-label__link">忘記密碼？</a>
        </label>
        <div class="input-wrapper">
          <input
            type="password"
            id="password"
            class="form-input"
            autocomplete="current-password"
            required
          >
          <button type="button" class="input-toggle" aria-label="顯示/隱藏密碼">
            👁
          </button>
        </div>
      </div>

      <button type="submit" class="btn btn--primary btn--full">
        登入
      </button>
    </form>

    <p class="auth-card__footer">
      還沒有帳號？<a href="/register">立即註冊</a>
    </p>
  </div>
</div>
```
