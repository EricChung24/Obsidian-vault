# 套件管理

> npm、yarn、pnpm 三大套件管理工具比較與用法。

← [[前端工程師]]

---

## 常用指令對照

| 操作 | npm | yarn | pnpm |
|------|-----|------|------|
| 安裝依賴 | `npm install` | `yarn` | `pnpm install` |
| 新增套件 | `npm i pkg` | `yarn add pkg` | `pnpm add pkg` |
| 新增開發依賴 | `npm i -D pkg` | `yarn add -D pkg` | `pnpm add -D pkg` |
| 全域安裝 | `npm i -g pkg` | `yarn global add pkg` | `pnpm add -g pkg` |
| 移除套件 | `npm uninstall pkg` | `yarn remove pkg` | `pnpm remove pkg` |
| 執行腳本 | `npm run dev` | `yarn dev` | `pnpm dev` |
| 更新套件 | `npm update` | `yarn upgrade` | `pnpm update` |
| 查看過時 | `npm outdated` | `yarn outdated` | `pnpm outdated` |

## package.json 重點欄位

```json
{
  "name": "my-app",
  "version": "1.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview",
    "test": "vitest",
    "lint": "eslint src"
  },
  "dependencies": {
    "react": "^18.3.0",
    "react-dom": "^18.3.0"
  },
  "devDependencies": {
    "@types/react": "^18.3.0",
    "typescript": "^5.4.0",
    "vite": "^5.2.0"
  },
  "engines": {
    "node": ">=20.0.0"
  }
}
```

## pnpm 優勢

- **磁碟效率**：硬連結共用 node_modules，省 60%+ 磁碟空間
- **速度最快**：安裝速度通常比 npm/yarn 快 2-3 倍
- **Monorepo**：內建 workspace 支援

```yaml
# pnpm-workspace.yaml
packages:
  - 'packages/*'
  - 'apps/*'
```

## 版本符號說明

```
"react": "18.3.0"   # 精確版本
"react": "~18.3.0"  # 允許 patch 更新 (18.3.x)
"react": "^18.3.0"  # 允許 minor 更新 (18.x.x)
"react": "*"        # 任意版本（危險！）
```
