# 🚀 前端工程師知識庫

這是一份完整的**前端工程師知識地圖**，使用 Obsidian 組織，涵蓋從基礎技術到進階應用的所有重要主題。無論你是初學者還是想深化技能的開發者，都能在這裡找到有用的學習資源。

---

## 📚 知識結構

本知識庫包含 **12 個主要學習模組**，共涵蓋超過 80 個核心主題：

### 01 基礎技術 (`01-基礎技術/`)
前端開發的核心基礎知識

- **HTML** - 語意化、表單結構、SEO 基礎
- **CSS** - 盒模型、選擇器、排版與樣式
- **CSS 動畫** - transition、animation、關鍵影格設計
- **JavaScript** - 語法、閉包、陣列方法、模組化
- **TypeScript** - 型別系統、泛型、實務型別設計
- **Event Loop** - Call Stack、Microtask、Task Queue
- **Async-Await 深度指南** - Promise 鏈、錯誤處理最佳實踐
- **瀏覽器渲染原理** - Reflow、Repaint、關鍵渲染路徑
- **切版技巧** - Flexbox、Grid、RWD 實務應用
- **正則表達式** - 模式匹配、表單驗證、字串處理

### 02 框架與函式庫 (`02-框架與函式庫/`)
現代前端主流框架和工具庫

- **React** - Hooks、元件設計、狀態管理
- **Vue** - Composition API、模板語法、元件通訊
- **Next.js** - SSR、SSG、App Router、資料取得策略
- **Nuxt** - Vue 生態中的 SSR/SSG 框架
- **React Router** - 路由設計、頁面切換、巢狀路由
- **React Query** - Server State 管理、快取與失效策略
- **表單處理** - React Hook Form、驗證流程、提交最佳實踐
- **i18n 國際化** - 多語系設計、翻譯流程、地域適配
- **PWA 漸進式網頁應用** - 離線支援、安裝體驗、快取策略
- **常用工具函式庫** - 日期處理、工具函式、資料操作

### 03 狀態管理 (`03-狀態管理/`)
不同工具的狀態管理方案

- **Redux** - Reducer 模式、Middleware、DevTools
- **Zustand** - 輕量級方案、簡化語法
- **Pinia** - Vue 3 官方推薦的狀態管理

### 04 樣式解決方案 (`04-樣式解決方案/`)
CSS 框架和預處理器

- **Tailwind CSS** - Utility-first、響應式設計、客製化
- **Sass** - 變數、混入、嵌套、模組化
- **CSS-in-JS** - 元件化樣式、運行時樣式注入
- **Bootstrap 5** - 柵欄系統、元件庫、主題客製化

### 05 建置工具 (`05-建置工具/`)
專案建置和開發環境設定

- **Vite** - 現代建置工具、快速開發伺服器
- **Webpack** - 舊專案維護、複雜打包配置
- **套件管理** - npm、yarn、pnpm 對比與最佳實踐
- **ESLint 與格式化** - 代碼質量控制、Prettier 整合

### 06 測試 (`06-測試/`)
測試策略和實施方案

- **單元測試** - Jest、Vitest、測試編寫
- **E2E 測試** - Cypress、Playwright、用戶行為模擬
- **測試策略** - 測試金字塔、覆蓋率目標

### 07 效能優化 (`07-效能優化/`)
Web 應用效能提升方案

- **前端快取策略** - 瀏覽器快取、HTTP 快取、CDN
- **渲染優化** - 虛擬滾動、按需載入、懶加載
- **程式碼分割** - 動態 import、路由分割、包體積優化
- **Core Web Vitals** - LCP、FID、CLS 優化
- **React 渲染與重渲染** - render 觸發來源、元件更新優化
- **SEO 優化** - Meta 標籤、結構化資料、爬蟲友善

### 08 瀏覽器與 Web APIs (`08-瀏覽器與Web-APIs/`)
瀏覽器 API 和網路相關技術

- **HTTP 協議** - 請求/響應、狀態碼、版本差異
- **網路協定** - TCP/IP、DNS、HTTPS、WebSocket
- **Fetch API** - 現代網路請求、流式傳輸
- **Axios** - HTTP 客戶端、攔截器、請求取消
- **CORS** - 跨域資源共享、預檢請求、認證
- **Cookie、Session、JWT** - 身份驗證方案、安全考量
- **Storage APIs** - localStorage、sessionStorage、IndexedDB
- **DOM 操作** - 選擇器、事件處理、效能考量
- **Canvas 與 SVG** - 2D 繪圖、向量圖形、動畫
- **API 錯誤處理實戰** - 錯誤重試、超時處理、用戶提示
- **Web Worker 與效能** - 多執行緒、長任務卸載

### 09 設計模式 (`09-設計模式/`)
程式碼設計和架構模式

- **JavaScript 設計模式** - 工廠模式、觀察者模式、策略模式
- **元件設計模式** - 高階元件 (HOC)、Render Props、Slots
- **React Hooks 模式** - 自訂 Hooks、Hooks 最佳實踐
- **SOLID 原則** - 單一職責、開放-封閉、依賴注入

### 10 無障礙與資安 (`10-無障礙與資安/`)
使用者體驗和安全防護

- **a11y 無障礙設計** - ARIA、鍵盤導航、螢幕閱讀器
- **前端資安** - XSS、CSRF、依賴安全、敏感資訊保護

### 11 DevOps 與部署 (`11-DevOps與部署/`)
應用部署和運維

- **Git** - 版本控制、分支策略、合併衝突解決
- **CI/CD** - 自動化測試、自動部署、Pipeline 設定
- **Docker 與容器化** - 容器化應用、映像構建、編排
- **前端監控** - 錯誤追蹤、效能監測、使用者行為分析
- **前端部署排錯實戰** - 常見問題、除錯技巧、灰度發布

### 12 面試準備 (`12-面試準備/`)
面試相關資源和準備指南

- **常見面試題** - 熱門問題精選
- **JavaScript 面試回答模板** - 標準解答方式
- **React 面試回答模板** - 框架相關深度問題
- **系統設計** - 架構設計、可擴展性
- **前端系統設計回答模板** - 設計實戰案例
- **演算法與資料結構** - 算法題目、資料結構應用

## 🎯 使用方式

### 作為 Obsidian 知識庫
1. 在 Obsidian 中開啟此資料夾
2. 查看主檔案 `前端工程師.md` 獲得完整索引
3. 利用連結導航在不同主題間探索
4. 使用 Obsidian 的搜尋功能快速定位內容

### 作為學習路徑

**初級開發者**
```
1. 從 01-基礎技術 開始，掌握 HTML、CSS、JavaScript 基礎
2. 學習 02-框架與函式庫 中的 React/Vue 基礎
3. 實踐 06-測試 中的單元測試編寫
4. 瞭解 11-DevOps與部署 中的 Git 和基本部署
```

**中級開發者**
```
1. 深入 09-設計模式 的各類設計模式
2. 掌握 03-狀態管理 的進階狀態方案
3. 學習 07-效能優化 的各項優化技巧
4. 研究 08-瀏覽器與Web-APIs 的深層原理
```

**高級開發者**
```
1. 系統設計和架構 (12-面試準備)
2. 效能優化實戰 (07-效能優化)
3. DevOps 完整流程 (11-DevOps與部署)
4. 資安與無障礙 (10-無障礙與資安)
```
