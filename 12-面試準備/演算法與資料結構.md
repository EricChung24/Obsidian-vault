# 演算法與資料結構

> 前端面試不一定每次都考演算法，但中大型公司很常用它來看你的基本邏輯能力。

> [[前端工程師]]

---

## 常見複雜度

```txt
O(1)
O(log n)
O(n)
O(n log n)
O(n^2)
```

面試不只是背公式，而是知道：

- 哪種寫法比較慢
- 哪裡可以換時間或空間

---

## 常見資料結構

- Array
- Object / HashMap
- Stack
- Queue
- Tree
- Graph

---

## 前端常見考法

- 括號是否合法
- 陣列去重
- 字串處理
- BFS / DFS 基礎題
- HashMap 題型

---

## 準備建議

- 先把 Array / HashMap 題型熟
- 再練 Stack / Queue
- 最後補 Tree / Graph
